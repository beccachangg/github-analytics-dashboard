# gitHub analytics dashboard

an interactive, real-time dashboard for visualizing gitHub repository metrics and trends. built with next.js, typescript, and tailwind css.

## features

- **commit activity heatmap** - visualize contribution patterns over the last 52 weeks
- **language distribution** - interactive pie chart showing codebase composition
- **contributor leaderboard** - ranking of top contributors with contribution counts
- **issue & pr velocity** - track issue and pull request trends over time
- **dark mode** - smooth dark/light theme toggle with persistence
- **responsive design** - works seamlessly on desktop, tablet, and mobile
- **real-time data** - fetches live data from github api
- **loading states** - skeleton loaders and smooth transitions
